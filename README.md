# Los Pollos Hermanos: Ночная смена

2D игра на **Windows Forms** в стиле пиксель-арта.

- Жанр: симулятор работника общепита
- Режим: одиночная игра
- Управление: клавиатура (`WASD`/стрелки + `E`)
- Длительность смены: 3-5 минут
- Архитектура: строгий `MVC`

## Что уже реализовано (Недели 1-2)

- solution и структура проекта (`src`, `tests`, `docs`)
- модель игрового мира (игрок, станции, заказы, таймер смены)
- полный цикл заказа:
  - принять заказ
  - выполнить этапы на станциях
  - выдать клиенту
- очки, рейтинг, штрафы и условия поражения (`уволен`)
- стартовый экран, минимальная обучалка, перезапуск смены
- unit-тесты на ключевые правила модели

Подробно: [docs/week-1-2-progress.md](docs/week-1-2-progress.md)

- .NET SDK 8+


```powershell

```

## Управление

- `Enter`/`Space` — старт смены или рестарт после конца
- `W`, `A`, `S`, `D` или стрелки — движение
- `E` — взаимодействие со станцией

## Правила MVP

1. Подойти к стойке заказа и принять заказ.
2. Выполнить этапы приготовления на нужных станциях.
3. Подойти к станции выдачи и отдать заказ.
4. За ошибки падает рейтинг и растет счетчик промахов.
5. Если ошибок слишком много, сотрудника увольняют.
6. Если дожить до конца таймера и не быть уволенным — смена пройдена.
